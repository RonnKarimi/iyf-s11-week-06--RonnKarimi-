This repository contains my solutions for **Week 6** of the IYF Software Engineering program.

- Understand synchronous and asynchronous JavaScript
- Create and use callbacks
- Work with JavaScript Promises
- Chain Promises
- Use Promise.all() and Promise.allSettled()

- Fetch data from REST APIs
- Send POST requests
- Display API data dynamically
- Store data using localStorage
- Build a complete weather application using an external API
